# SchiferQRX

SchiferQRX — это веб-приложение на Django для создания и извлечения двухслойных QR-кодов с использованием LSB-стеганографии. Проект включает систему аутентификации пользователей, личные кабинеты, статистику активности, систему достижений и поддержку ограничения времени жизни скрытого сообщения.

> 🏆 Проект занял **1 место на Сократовских чтениях** (секция информационных технологий, Московский международный университет, апрель 2026).

## Возможности

- регистрация и авторизация пользователей;
- генерация QR-кодов с открытым и скрытым текстом (двухслойное кодирование);
- извлечение скрытого текста из специально созданных QR-кодов;
- поддержка метаданных в формате JSON (версия, срок действия);
- ограничение времени жизни скрытого сообщения (от 1 минуты до 30 дней);
- личный кабинет с отображением статистики (количество генераций, декодирований, скачиваний);
- график активности пользователя (кольцевая диаграмма);
- система достижений (ачивки) и история действий;
- администрирование через встроенную панель Django;
- готовая контейнеризация (Docker, docker-compose) для развёртывания на сервере.

## Стек технологий

- Python 3.9
- Django 3.2.16
- PostgreSQL 13 (в продакшене), SQLite3 (для разработки)
- Gunicorn
- Nginx (reverse proxy + статика)
- Docker, docker-compose
- HTML, CSS, JavaScript, Chart.js
- Библиотеки: qrcode, Pillow, whitenoise, dj-database-url, psycopg2-binary
